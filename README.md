### 🚀 사용 방법 (How to use)
1. 이 리포지토리를 **Fork**하거나, 상단의 **[Use this template]** 버튼을 눌러 내 리포지토리로 복제합니다.
2. Chrome 웹 스토어에서 **[BaekjoonHub]** 확장 프로그램을 설치합니다.
3. 확장 프로그램 설정에서 **Authenticate**를 진행하고, 방금 만든 리포지토리를 선택하여 연동합니다.
4. 리포지토리의 `Settings > Actions > General`로 이동하여 **Workflow permissions**를 **Read and write permissions**로 변경하고 저장합니다.
5. 이제 백준에서 문제를 풀면 자동으로 코드가 업로드되고, **Issue 탭에 풀이 노트가 자동 생성**됩니다.
